## Задание 4

### Описание
Разработать ассемблер и интерпретатор для учебной виртуальной машины
(УВМ). Система команд УВМ представлена далее.
Для ассемблера необходимо разработать читаемое представление команд
УВМ. Ассемблер принимает на вход файл с текстом исходной программы, путь к
которой задается из командной строки. Результатом работы ассемблера является
бинарный файл в виде последовательности байт, путь к которому задается из
командной строки. Дополнительный ключ командной строки задает путь к файлу-логу, в котором хранятся ассемблированные инструкции в духе списков
“ключ=значение”, как в приведенных далее тестах.
Интерпретатор принимает на вход бинарный файл, выполняет команды УВМ
и сохраняет в файле-результате значения из диапазона памяти УВМ. Диапазон
также указывается из командной строки.
Форматом для файла-лога и файла-результата является csv.
Необходимо реализовать приведенные тесты для всех команд, а также
написать и отладить тестовую программу

Система команд УВМ представлена ниже. Необходимо разработать ассемблер и интерпретатор для выполнения этих команд. Описание работы с каждым из компонентов, а также тесты для команд, представлены ниже.

### Описание системы команд УВМ

#### 1. Загрузка константы
| A  | B  | C  |
|----|----|----|
| 0—3 | 4—18 | 19—50 |
| 2  | Адрес | Константа |

**Размер команды:** 7 байт.

**Операнд:** поле C.

**Результат:** значение в памяти по адресу, который задан полем B.

#### 2. Чтение значения из памяти
| A  | B  | C  |
|----|----|----|
| 0—3 | 4—18 | 19—33 |
| 14 | Адрес | Адрес |

**Размер команды:** 5 байт.

**Операнд:** значение в памяти по адресу, который задан полем B.

**Результат:** значение в памяти по адресу, который задан полем C.

#### 3. Запись значения в память
| A  | B  | C  | D  |
|----|----|----|----|
| 0—3 | 4—10 | 11—25 | 26—40 |
| 10 | Смещение | Адрес | Адрес |

**Размер команды:** 6 байт.

**Операнд:** значение в памяти по адресу, который задан полем C.

**Результат:** значение в памяти по адресу, который равен сумме адреса (значение в памяти по адресу, который задан полем D) и смещения (поле B).

#### 4. Бинарная операция: побитовый арифметический сдвиг вправо
| A  | B  | C  | D  |
|----|----|----|----|
| 0—3 | 4—18 | 19—33 | 34—48 |
| 15 | Адрес | Адрес | Адрес |

**Размер команды:** 7 байт.

**Первый операнд:** значение в памяти по адресу, который задан полем C.

**Второй операнд:** значение в памяти по адресу, который задан полем D.

**Результат:** значение в памяти по адресу, который задан полем B.

### Тестовая программа
Выполнить поэлементно операцию побитовый арифметический сдвиг вправо над
двумя векторами длины 7. Результат записать в первый вектор.
